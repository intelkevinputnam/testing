Step 1: Download and Install the Package

In this step, you will download and unpack the Intel® Time Coordinated Computing Tools (Intel® TCC Tools) package.

  1. If you haven’t already downloaded the package, go to the Download page. Click Free Download for Linux*.

  2. Make the package shell script executable:

    
chmod +x ./l_tcc_tools_b_<version>_[offline].sh

  3. On your host system, extract the installer into the current directory. Replace <version> as appropriate.

    
./l_tcc_tools_b_<version>_[offline].sh -x -f .
    Note:

    You may not see any output on the screen for about 30 seconds.

  4. Go to the directory containing the extracted files:

    
cd l_tcc_tools_b_<version>_[offline]

  5. Make the install shell script executable:

    
chmod +x ./install.sh

  6. Run the following command to install the files. By default, the files will be installed to the /home/<user>/intel/oneapi/tcc_tools/latest directory. You can specify a different install directory by using the --install-dir option.

    
./install.sh\
                           --action=install\
                           --components=all\
                           --eula=accept\
                           --intel-sw-improvement-program-consent=no\
                           --silent

    Or

    
./install.sh\
                           --action=install\
                           --components=all\
                           --eula=accept\
                           --intel-sw-improvement-program-consent=no\
                           --silent\
                           --install-dir=<install_directory>
